Password manager application using **React-Redux**, **Node**, **Typescript**, and **Postgres**.

## Project setup

1. Clone repo into local machine

```bash
git clone https://github.com/thaivu93/password-manager.git
cd password-manager
```

2. Install dependencies

-   **Backend**

```bash
# Backend
cd backend
npm install
```

-   **Front end**

```bash
# Frontend
cd frontend
npm install

```
